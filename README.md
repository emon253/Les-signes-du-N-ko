# Les signes du N'ko

A game to discover and learn the basics of the N'Ko script: explore its origins, writing direction, and key features. Created by Hervé Yvis, this game is released under the [Creative Commons Attribution 4.0 International (CC BY 4.0) license](https://creativecommons.org/licenses/by/4.0/deed).

The game EGF is embeded using Embedded EGF 1.1 Reader.
